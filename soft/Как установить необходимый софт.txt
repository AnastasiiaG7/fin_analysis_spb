Для реализации проекта необходимо выполнить следующие шаги:

1. Установить Anaconda. Anaconda — дистрибутив языков программирования Python и R. В Anaconda находится интерактивную среду Jupyter Notebook.
Ссылка на установку: https://www.anaconda.com/download 
Также можно использовать другую среду, например Google Colab, который не требует предварительной установки.

2. Установить DBeaver. DBeaver - приложение, предназначенное для управления базами данных
Ссылка на установку: https://dbeaver.io/download/ 

3. Ссылка на СПАРК интерфакс: https://spark-interfax.ru/
